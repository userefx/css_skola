Introduktion till CSS

CSS står för Cascading Style Sheets. Det är språket vi använder för att ge webbsidor färg, form och layout.

CSS Syntax

En CSS-regel består av en selector och ett declaration block.

h1 {
  color: blue;
  font-size: 24px;
}

I exemplet ovan är h1 selektorn, medan color är egenskapen (property) och blue är värdet (value).

De tre huvudselektorerna

1. Element-selektor

Väljer alla element av en viss sort. Bra för globala inställningar.

p { color: grey; }
Target: Alla <p> element på sidan.

2. Class-selektor (.)

Används för att styla specifika element. Du kan använda samma klass på många olika element.

.highlight { background-color: yellow; }
HTML: <p class="highlight">Viktig text</p>

3. ID-selektor (#)

Används för att identifiera ett unikt element på hela sidan. Ett ID får bara användas en gång per sida.

#main-header { font-size: 40px; }
HTML: <h1 id="main-header">Välkommen</h1>

Pro-tip: Specificitet

Om ett element har både en klass och ett ID som ger olika färger, kommer ID-selektorn alltid att vinna eftersom den är mer specifik!
